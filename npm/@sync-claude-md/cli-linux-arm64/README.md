# sync-claude-md binary for Linux ARM64

This is a platform-specific binary package for [sync-claude-md](https://github.com/lohn/sync-claude-md).

You probably don't want to install this package directly. Instead, install the main package:

```bash
npm install --save-dev sync-claude-md
```

For more information, see the [sync-claude-md repository](https://github.com/lohn/sync-claude-md).
